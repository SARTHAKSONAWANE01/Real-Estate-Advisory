## ✦ Technical Architecture & Styling

The landing platform is built using a highly optimized, framework-free modern web stack designed to maximize responsiveness, load speed, and SEO visibility.

### 1. Typography & Aesthetic
*   **Luxury Editorial Serif**: `Cormorant Garamond` is applied to headings to provide an elegant, bespoke feel.
*   **Modern Geometric Sans**: `DM Sans` is used for highly legible, premium layout body text.
*   **Color System**: Built around bespoke CSS Custom Properties (`--cream`, `--warm-white`, `--gold`, `--ink`, `--green-accent`) reflecting standard luxury real estate styling.

### 2. Interaction Design (Vanilla CSS & JavaScript)
*   **Scroll-Driven Animations**: Leverages a highly efficient `IntersectionObserver` in JavaScript to trigger smooth fade-ups as elements enter the viewport.
*   **Responsive Navigation System**: Automatic transition of navbar padding on scroll, coupled with an accessible, keyboard-friendly slide-in mobile navigation menu.
*   **Bespoke Keyframe Animations**: Custom infinite rotations (stamp badge), slow-spin concentric orbital patterns, scroll pulses, and dynamic typography marquees.

### 3. SEO & Semantic HTML5 Best Practices
*   Fitted with optimized meta tags, OpenGraph properties, theme color, and Twitter Card specifications.
*   Strict heading hierarchy with a semantic layout (`nav`, `section`, `footer`).
*   Fully optimized for both ultra-wide screens and mobile viewports.

## ✦ Local Development & Deployment

The codebase is designed with maximum portability in mind—zero compilation, zero build steps, and zero external framework overhead.

### Just Open & Run
1.  Clone this repository:
    ```bash
    git clone https://github.com/SARTHAKSONAWANE01/Real-Estate-Advisory.git
    cd Real-Estate-Advisory
    ```
2.  To view the site, simply open `index.html` in your browser.
3.  For a fully-featured local testing environment (enabling standard routing and local absolute path structures), run a lightweight web server:
    ```bash
    # Using Node's serve utility
    npx serve .
    
    # Or using Python's built-in HTTP server
    python -m http.server 8000
    ```
